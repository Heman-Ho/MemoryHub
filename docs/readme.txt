Instructions to Run code: 
1. Go to Tools-> Firebase
2. Find Crashlytics and click on "Get Started with Firebase Crashlytics"
3. Make a firebase account and tell Heman (lhh4@sfu.ca) to add you to the project
4. Click the "connect to Firebase" button
5. Click the "Add Crashlytics SDK and plugin to your app" button
6. You should be able to run the code now
